DDS is the Data platform of HIGHLANDER project.

The purpose of the Highlander Data platform is to provide users with a single point of access to the results of the Highlander project. The results consist of the climate datasets and the services based on them, indicated with the acronym DApOS.

The Highlander Data Portal is essentially the website that provides this access point, the web frontend that allows users to interact with the Highlander Data Platform through a broad array of services

The features currently present on DDS starting from the MAIN MENU items are:

  • DATASETS: catalog of data present on the platform. ACCESS IS PUBLIC - Download function. LOGIN VIA ACCOUNT
  • APPLICATION DATA: visualizations for DApOS. . ACCESS IS PUBLIC
  • MY REQUEST: list of data requests made by the user. LOGIN VIA ACCOUNT

The DATA is open, the indication of the license is present on the sheet of each dataset.
