The HIGHLANDER Data Delivery System (DDS)

The DDS is the Data Portal of the HIGHLANDER project.

The purpose of the HIGHLANDER Data Portal is to provide users with a single point of access to the results of the HIGHLANDER project. The results consist of the new high resolution climate datasets and the downstream applications and (pre-)operational services based on them.

The features currently available in the DDS are:

  • Public section:

    • DATASETS: the catalog of datasets
    • APPLICATIONS: visualisation tools

  • Private section :

    (an account is needed)
    • Get Data: compose a request to filter and crop a dataset and submit the request
    • MY REQUESTS: download data resulting from your requests

The indication of the licenses is shown in the dataset detail page.
